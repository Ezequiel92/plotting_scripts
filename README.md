Scripts to make plots from the data of GADGET3 simulations. 

All use of the module [GADGETPlotting.jl](https://github.com/Ezequiel92/GADGETPlotting). 

## Folders

The figures, GIFs and videos generated will be saved in `../plots/XXX/` (the folder will be created if it doesn't exist) where XXX is the number of the script. The snapshots are expected to be located in `../../sim_data/isolated/run_00/`, `../../sim_data/cosmological/run_A_01/`, etc. So, the expected file structure is:

    .
    ├── sim_data
    │    ├── isolated
	│    │    ├── run_00
	│    │    ├── run_A_01
	│	 │	  └── ...
	│    ├── cosmological
	│    │    ├── run_00
	│    │    ├── run_A_01
	│	 │	  └── ...
    │    └── ...
    ├── plots 
    │    ├── 001
    │    │    ├── ...
    │    │    └── ...
    │    ├── 002
    │    │    ├── ...
    │    │    └── ...
    │    └── ...
    └── code
         ├── GADGETPlotting 
         │    ├── src 
         │    │    ├── GADGETPlotting.jl   
         │    │    └── ...  
         │    └── ...
         └── plotting_scripts
              ├── plots_001.jl
              ├── plots_002.jl
              └── ...
         

## Documentation

A comment at the beginning of each script explains which figures are produced.

## Warning

- These scripts are written for data generated by GADGET3 (which is not a publicly available code) and require snapshot files which are too heavy for a GitHub repository, 
- These scripts are written mainly for personal use and may break at any moment. So, no guaranties are given.
